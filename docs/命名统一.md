## 1. 四层语义（必守）

| 层 | 标识 | 典型位置 |
| --- | --- | --- |
| 法币 | `amount` + `currency` | `company_recharge_orders` |
| 内部 point | `point` / `*Point` | `wallet_remain`、`budget_consumed.consumed`、`usage_ledger.amount` |
| 组织预算 limit | `budget` / `personalBudget` | `org_nodes.budget`、`members.personal_budget`、`platform_keys.budget` |
| NewAPI 通道 | `newAPIUnits`；协议 `quota` | 仅 `integration/newapi`、`adminport`、ingest log |

**刻意保留的双名（不改 API）：**

- UI 中文可继续用「额度」
- 域类 `BudgetProjector`（预算 consumed 投影，非 Project CRUD）

组织轴 JSON 已消耗字段统一为 **`consumed`**（`PlatformKey`、`MemberBudget`、`MemberBudgetSummary`、`BudgetNode`、`Project`）。

**实体边界：**

- **Project**（US-07）：`Project` / `projectId` / `ownerDepartmentId` / UI 层 `ProjectView`；禁止 `budgetGroup*`
- **US-11 用途**：`appName` / `usageScope`（本期不做 `scope: project_member`）

## 6. 维护约定

1. 组织轴 JSON **禁止**新增 `quota` 或 `budgetGroup*`。
2. NewAPI 集成：`quota` 仅限 `integration/newapi` + `adminport` + ingest 解析。
3. 新功能：项目池 → `Project` / `projectId`；用途 → `appName`。
4. 分配页局部变量：`departmentBudget` / `projectBudget`；实体 JSON 仍用 `budget`。
